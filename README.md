# FilterX

FilterX is a university project developed collaboratively with friends as part of our academic studies.  
The project was built using React for the frontend and Spring Boot for the backend.

## Technologies Used

- React
- Spring Boot
- Java
- MySQL
- REST API
- HTML/CSS/JavaScript

## About the Project

FilterX was created for learning and educational purposes as a collaborative university project.
The project focuses on content filtering and classification concepts. 
However, the filtering model is currently very limited because it was trained locally with a small dataset and limited resources.
As a result, the filtering accuracy and overall performance are not yet reliable for real-world production use.

## Current Limitations

- Limited training dataset
- Locally trained filtering model
- Weak filtering accuracy
- Basic UI/UX
- Needs optimization and refactoring
